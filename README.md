# Перевод Modern Quantum Chemistry

Репозиторий содержит русскоязычный перевод книги *Modern Quantum
Chemistry* в формате Typst. 

## Содержание

- `main.typ` - точка входа документа; здесь задается порядок включения
  глав.
- `titlepage.typ` - титульная страница.
- `settings.typ` и `macros.typ` - общие настройки оформления, шаблон и
  макросы.
- `chapters/` - текст перевода по главам и разделам.
- `figures/` - изображения, используемые в тексте.
- `cetz/` - исходники схем и рисунков CeTZ.
- `assets/` - дополнительные исходные материалы.
- `typst.toml` - метаданные пакета и версия компилятора Typst.
- `Makefile` - команды форматирования, проверки и сборки.

## Требования

- Typst `0.15.0`.
- `typstyle` для форматирования Typst-файлов.
- `make` для запуска стандартных команд.

## Команды

```sh
make fmt        # отформатировать файлы
make fmt-check  # проверить форматирование
make build      # собрать main.pdf
make ci         # проверить форматирование и собрать PDF
```

## Порядок работы

1. Найти нужный раздел в `chapters/` или создать новый `.typ`-файл.
2. Если раздел новый, подключить его через `#include` в родительской
   главе или в `main.typ`.
3. Рисунки класть в `figures/`, исходники схем - в `cetz/`.
4. Для общих обозначений и повторяющихся конструкций использовать
   `macros.typ`, не дублируя макросы в главах.
5. Перед отправкой изменений выполнить `make fmt`, затем `make ci`.
6. Коммитить исходники перевода и рисунков. Сгенерированный `main.pdf`
   не добавлять в изменения.
